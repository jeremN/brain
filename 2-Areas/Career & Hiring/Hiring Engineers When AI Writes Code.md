# Hiring Engineers When AI Writes Code

**Source:** [https://www.tolans.com/relay/how-we-hire-engineers-when-ai-writes-our-code](https://www.tolans.com/relay/how-we-hire-engineers-when-ai-writes-our-code)
**Category:** Career & Hiring

## Summary
An analysis of how AI code generation changes engineering hiring practices and skills evaluation. Discusses what to look for in candidates when AI handles routine coding tasks.
## Why It's Interesting
Critical guide for adapting hiring practices to the AI-augmented development landscape.
## Key Takeaways
- Focus shifts from coding speed to architecture and design thinking
- Ability to work with AI tools becomes essential skill
- Problem decomposition skills become more valuable
- Communication and collaboration skills increase in importance
