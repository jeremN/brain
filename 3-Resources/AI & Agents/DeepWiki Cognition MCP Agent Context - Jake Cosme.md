# DeepWiki + Cognition MCP pour un Contexte d'Agent Illimité

**Source:** [@jake__cosme](https://x.com/jake__cosme/status/2040081941337448558)
**Category:** Architecture d'Agents

## Résumé
Jake Cosme propose une combinaison puissante pour créer des agents avec un contexte pratiquement illimité : un repository GitHub gratuit + deepwiki.com + Cognition MCP. Cette approche s'inspire des réflexions de Karpathy sur les bases de connaissances pour les LLMs et offre une solution accessible pour augmenter la capacité contextuelle des agents.

## Pourquoi c'est intéressant
Cette combinaison d'outils libres et gratuits permet aux développeurs de créer des agents sophistiqués avec accès à des connaissances étendues sans coûts prohibitifs. C'est une évolution importante dans la démocratisation des agents IA avancés.

## Points clés
- **Repository GitHub gratuit** comme stockage central
- **deepwiki.com** pour organiser et servir les connaissances
- **Cognition MCP** pour l'intégration avec les agents
- **Bases de connaissances LLM** inspirées par Karpathy
- **Contexte illimité** pour les agents IA
