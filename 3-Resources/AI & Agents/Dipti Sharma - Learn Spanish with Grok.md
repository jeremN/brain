# Learn Spanish with Grok

**Source:** [https://x.com/Diptish09/status/2000915320564015218](https://x.com/Diptish09/status/2000915320564015218)
**Category:** AI & Agents

## Résumé
Dipti Sharma a supprimé Duolingo après avoir appris l'espagnol en 37 jours avec Grok. Elle partage six prompts efficaces pour utiliser Grok comme alternative pour l'apprentissage des langues, montrant comment les LLMs peuvent remplacer les applications traditionnelles de langage.

## Pourquoi c'est intéressant
C'est un retour d'expérience impressionnant qui montre comment les LLMs peuvent être plus efficaces que les applications spécialisées traditionnelles. Les prompts partagés offrent une base pratique pour d'autres cherchant à apprendre des langues avec Grok.
