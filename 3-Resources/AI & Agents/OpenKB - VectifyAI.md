# OpenKB - VectifyAI

**Source:** [github.com/VectifyAI/OpenKB](https://github.com/VectifyAI/OpenKB)
**Category:** AI & Agents

## Summary
OpenKB is an open source knowledge base system from VectifyAI using vector search and embeddings. Enables AI agents to efficiently store, retrieve, and reason over large document and knowledge bases.

## Why It's Interesting
Priority topic: provides scalable knowledge retrieval infrastructure for agents to augment reasoning with large document corpuses.

## Key Takeaways
- Vector-based knowledge base storage
- Semantic search capabilities for agents
- Scalable document retrieval
- Open source knowledge infrastructure
