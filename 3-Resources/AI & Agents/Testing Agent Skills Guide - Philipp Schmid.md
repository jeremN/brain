# Guide de Test des Skills d'Agent

**Source:** [@_philschmid](https://x.com/_philschmid/status/2029570052530360719)
**Category:** AI & Agents

## Résumé
Les agent skills sont puissants mais souvent générés par l'IA et non testés. Philipp Schmid offre un guide pratique: définir des critères de succès clairs, puis créer 10-12 prompts avec des vérifications déterministes pour valider chaque skill. Avec 694 likes et 865 bookmarks, c'est un guide essentiel pour la qualité des agents.

## Pourquoi c'est intéressant
La reconnaissance que les agent skills générés par l'IA ont souvent des problèmes de qualité est honnête et utile. La méthodologie proposée (10-12 prompts avec vérifications) est concrète et reproductible, offrant une solution pratique.

## Points clés
- **Problème de qualité**: Les agent skills générés par l'IA ne sont pas toujours testés
- **Critères de succès**: Définition claire des objectifs avant le test
- **Méthodologie concrète**: 10-12 prompts avec vérifications déterministes
