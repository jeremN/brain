# Thoth — Personal AI Sovereignty

**Source:** [https://github.com/siddsachar/Thoth](https://github.com/siddsachar/Thoth)
**Category:** AI & Agents

## Summary
Assistant IA local-first construit pour la souveraineté IA personnelle. Combine un agent ReAct (LangGraph) avec 25 outils/70 sous-opérations, knowledge graph, voice I/O, browser automation, et système de plugins — le tout exécutable offline via Ollama.

## Why It's Interesting
L'assistant IA personnel le plus complet en open source — knowledge graph avec "Dream Cycle" nocturne, voice natif, browser autonome, et Telegram intégré. Le tout local-first.

## Key Takeaways
- **ReAct Agent** : LangGraph + 25 outils/70 sous-opérations + token streaming
- **Knowledge Graph** : 10 types d'entités, recherche sémantique, explorateur visuel
- **Dream Cycle** : raffinement nocturne automatique (fusion de doublons, inférence de relations)
- **Voice I/O** : faster-whisper (STT) + Kokoro (TTS, 10 voix)
- **Browser Automation** : navigation autonome en Chromium visible
- **Telegram** : intégration complète avec support média
- **Plugins** : sandboxés, hot-reloadable, marketplace
- **Stack** : Python, Streamlit/NiceGUI, SQLite/FAISS, Ollama
