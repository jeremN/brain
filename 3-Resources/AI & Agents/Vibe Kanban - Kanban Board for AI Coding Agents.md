# Vibe Kanban - Kanban Board for AI Coding Agents

**Source:** [github.com/BloopAI/vibe-kanban](https://github.com/BloopAI/vibe-kanban)
**Category:** AI & Agents

## Summary

Vibe Kanban est un outil d'orchestration visuelle qui transforme le développeur en "manager d'agents IA". Il se présente comme un Kanban board local où tu crées des tâches, les assignes à des coding agents (Claude Code, Codex, Gemini CLI, Cursor, Copilot, etc.), et chaque agent travaille dans son propre Git worktree isolé — zéro conflit entre agents. Review de code avec diffs et commentaires inline, preview intégrée, puis merge via PR.

## Why It's Interesting

Approche la plus intuitive de l'orchestration multi-agents : reprend un workflow que les devs connaissent (Kanban + PR review) et y colle des agents en parallèle. Zero config, zero cost (tu paies juste les agents sous-jacents). Stack Rust+TypeScript solide, 23.7k stars.

## Key Takeaways

- Orchestration visuelle de 10+ coding agents en parallèle sur un Kanban board
- Isolation par Git worktrees — chaque agent a sa branche, zéro conflit
- Code review intégrée avec diffs et commentaires inline
- Preview d'application avec browser et devtools built-in
- Zero config : pointer un repo, créer des tâches, assigner des agents
- Apache 2.0, 23.7k stars, 272 releases — activement maintenu
