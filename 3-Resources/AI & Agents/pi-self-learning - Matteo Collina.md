# pi-self-learning — Memory for Coding Agents by Matteo Collina

**Source:** [https://x.com/matteocollina/status/2032140579899846765](https://x.com/matteocollina/status/2032140579899846765)
**Category:** AI & Agents

## Summary
Extension "pi" par Matteo Collina (créateur de Fastify, Node.js TSC) qui donne une vraie mémoire aux coding agents. Extrait ce qui a mal tourné après chaque tâche, score les apprentissages par fréquence/récence, et les injecte automatiquement dans les futures sessions.

## Why It's Interesting
Solution élégante au problème de la répétition d'erreurs par les agents IA — scoring par fréquence/récence pour ne garder que les leçons pertinentes. Par un des core contributors Node.js.

## Key Takeaways
- **Extraction automatique** des erreurs après chaque tâche
- **Scoring** par fréquence et récence
- **Injection automatique** des learnings dans les futures sessions
- **Extension pi** (platformatic) — pas besoin de changer d'agent
