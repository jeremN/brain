# Vibe Coded Ship Selection UI for Space Game

**Source:** [@DilumSanjaya on X](https://x.com/DilumSanjaya/status/2016193959408836932)
**Category:** AI & Agents

## Résumé
Dilum Sanjaya démontre la création d'une interface de sélection de vaisseaux pour un jeu spatial en utilisant le vibe coding. Il combine Nano Banana, Midjourney et Hugging Face pour générer le contenu.

## Pourquoi c'est intéressant
Ce projet montre comment les outils d'IA modernes peuvent être combinés pour créer rapidement des éléments d'interface de jeu professionnels, réduisant considérablement le temps de développement créatif.

## Points clés
- **Vibe coding** : Génération créative guidée par description
- **Multi-tool integration** : Nano Banana + Midjourney + Hugging Face
- **Game development** : UI creation pour les jeux spatiaux
