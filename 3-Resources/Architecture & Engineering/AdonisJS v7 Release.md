# AdonisJS v7 Is Here

**Source:** [https://adonisjs.com/blog/v7](https://adonisjs.com/blog/v7)
**Category:** Architecture & Engineering

## Summary
Release officielle d'AdonisJS v7 (25 février 2026). Upgrade incrémental depuis v6 avec peu de breaking changes mais des améliorations significatives : type safety end-to-end, nouveaux starter kits, et support Node.js 24+.

## Why It's Interesting
Framework Node.js/TypeScript full-stack avec une approche opinionated à la Laravel — la v7 apporte la type safety complète et une DX modernisée.

## Key Takeaways
- **45+ packages officiels** mis à jour, 3 nouveaux (@adonisjs/otel, @adonisjs/content, edge-markdown)
- **Type safety end-to-end** : routes, sérialisation, API clients
- **Node.js 24+** requis
- **Zero-config OpenTelemetry** intégré
- **Migration en 30-60 min** depuis v6
- **Nouveaux starter kits** avec authentification intégrée
