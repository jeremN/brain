# Projet 35k€ Stack Technique

**Source:** [https://x.com/NicolasCDev/status/1960960773552820478](https://x.com/NicolasCDev/status/1960960773552820478)
**Category:** Architecture & Engineering

## Résumé
Nicolas expose la stack technique d'un projet de 35k€ développé en 2 mois. La stack comprend React, React Native, et Firebase/Firestore, montrant comment ces technologies travaillent ensemble dans un projet réel de taille moyenne.

## Pourquoi c'est intéressant
Voir la stack technique d'un projet réel avec un budget et un calendrier définis offre des insights pratiques sur les choix technologiques. Cette transparence aide les autres à faire des décisions éclairées sur leurs propres projets.

## Points clés
- **Stack React + React Native + Firebase** pour un projet multi-plateforme
- **Budget et délai définis** : 35k€ en 2 mois
- **Démonstration pratique** des technologies en production
- **Transparence utile** pour les décisions architecturales
