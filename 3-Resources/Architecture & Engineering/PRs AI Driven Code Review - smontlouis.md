# Les PRs à l'Ère des LLM: Révision de Code Pilotée par l'IA

**Source:** [@_smontlouis](https://x.com/_smontlouis/status/2026560918180565450)
**Category:** Architecture & Engineering

## Résumé
"Depuis quelques temps je bloque sur les PRs à l'ère des LLM. Les PRs AI driven sont assez huge." smontlouis partage sa réflexion sur le défi de réviser et d'approuver les pull requests générées par l'IA. C'est une discussion importante sur comment les processus d'ingénierie doivent évoluer.

## Pourquoi c'est intéressant
Cette réflexion soulève un dilemme réel: comment les équipes d'ingénierie doivent-elles adapter leurs processus de code review face au code généré par l'IA en volume? C'est une question stratégique pour l'adoption du code assisté par l'IA.

## Points clés
- **Défi d'échelle**: Les PRs générées par l'IA en volume dépassent la capacité de révision
- **Processus d'ingénierie**: Besoin d'adapter les workflows traditionnels
- **Question d'adoption**: Comment intégrer le code IA au pipeline de développement
