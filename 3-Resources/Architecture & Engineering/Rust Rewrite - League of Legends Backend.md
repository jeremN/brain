## Summary
Documentation of Riot Games' effort to rewrite League of Legends backend services in Rust. Details motivation, challenges, and lessons learned from this major infrastructure migration.
## Why It's Interesting
High-profile case study of using Rust to modernize large-scale game infrastructure.

- Significant performance improvements possible

- Migration requires careful planning and execution
