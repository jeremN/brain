# CPUs Getting Worse: Silent Data Corruptions (SDCs) Are Now a Real Problem

**Source:** [@lauriewired on X](https://x.com/lauriewired/status/2022387879390122049)
**Category:** Architecture & Engineering

## Résumé
Laurie présente l'émergence des "Silent Data Corruptions" (SDCs) comme un problème réel dans les CPUs modernes. C'est une perspective inquiétante sur comment la fiabilité du hardware devient un facteur limitant croissant.

## Pourquoi c'est intéressant
Alors que nous optimisons logiciellement, la fiabilité matérielle se dégrade. C'est un problème systémique qui affecte tous les systèmes et demande une attention architecturale nouvelle pour les applications critiques.

## Points clés
- **Silent data corruptions** - Erreurs impossibles à détecter localement
- **Hardware reliability** - Dégradation de la qualité CPU
- **System impact** - Effets sur tous les niveaux d'abstraction
- **Architecture implications** - Nécessité de nouvelles approches
