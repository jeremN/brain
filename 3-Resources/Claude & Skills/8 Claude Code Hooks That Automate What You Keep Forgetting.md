# 8 Claude Code Hooks That Automate What You Keep Forgetting

**Source:** [https://x.com/santtiagom_/status/2040553002856005650](https://x.com/santtiagom_/status/2040553002856005650) (article by @zodchiii)
**Category:** Claude & Skills

## Summary
Article sur les Claude Code Hooks — le problème de CLAUDE.md est que c'est du contexte dans le prompt, pas une règle (le modèle peut l'ignorer). Les Hooks résolvent ça : ce sont des bouts de code qui s'exécutent toujours, peu importe ce que Claude décide. 8 hooks concrets pour automatiser ce qu'on oublie toujours (formater, ne pas toucher à certains fichiers, lancer les tests avant de finir).

## Why It's Interesting
Distinction cruciale entre contexte (ignorable) et règle (enforced). Les hooks transforment les souhaits en contraintes exécutables.

## Key Takeaways
- **CLAUDE.md = contexte** (le modèle peut l'ignorer) vs **Hooks = règle** (exécution garantie)
- **2 types principaux** de hooks
- **Cas d'usage** : formatage auto, protection de fichiers, tests obligatoires avant fin de tâche
