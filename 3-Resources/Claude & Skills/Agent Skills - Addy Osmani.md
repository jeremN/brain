# Agent Skills — Production-Grade Engineering Skills for AI Agents

**Source:** [https://github.com/addyosmani/agent-skills](https://github.com/addyosmani/agent-skills)
**Category:** Claude & Skills

## Summary
Collection de skills production-grade par Addy Osmani encodant les workflows, quality gates et best practices des ingénieurs seniors dans des fichiers Markdown structurés utilisables par les agents IA.

## Why It's Interesting
19 skills couvrant tout le cycle de dev (Define → Plan → Build → Verify → Review → Ship), avec 3 personas spécialisées et 4 checklists de référence. Compatible Claude Code, Cursor, Gemini CLI, Windsurf et GitHub Copilot.

## Key Takeaways
- **7 slash commands** : `/spec`, `/plan`, `/build`, `/test`, `/review`, `/code-simplify`, `/ship`
- **19 skills structurées** en 6 phases du cycle de dev
- **3 personas agents** : code reviewer, test engineer, security auditor
- **4 checklists** : testing, security, performance, accessibility
- **Multi-outil** : Claude Code, Cursor, Gemini CLI, Windsurf, GitHub Copilot
- **Distribution** : fichiers Markdown pouvant servir de system prompts
