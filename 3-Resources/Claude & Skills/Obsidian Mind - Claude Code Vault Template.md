# Obsidian Mind — Makes Claude Code Remember Everything

**Source:** [https://github.com/breferrari/obsidian-mind](https://github.com/breferrari/obsidian-mind)
**Category:** Claude & Skills

## Summary
Template de vault Obsidian conçu pour les ingénieurs utilisant Claude Code comme partenaire de réflexion. Injecte automatiquement les objectifs North Star, projets actifs, changements récents et contexte de tâches au début de chaque session.

## Why It's Interesting
Transforme Obsidian en knowledge graph queryable plutôt qu'en documents isolés — "folders group by purpose; links group by meaning". 15 slash commands et 9 subagents spécialisés.

## Key Takeaways
- **Session Memory** : injection automatique du contexte au démarrage
- **Structure** : projets, incidents, 1:1, décisions avec linking automatique
- **15 slash commands** : `/standup`, `/dump`, `/wrap-up`, `/review-brief`...
- **9 subagents** : analyse d'incidents, peer scanning, audit de vault
- **Performance tracking** : framework de compétences avec backlinks vers les preuves
- **Database views dynamiques** : dashboards auto-mis à jour
- **Recherche sémantique** : support QMD optionnel
- **Migration** : `/vault-upgrade` pour importer un vault existant
