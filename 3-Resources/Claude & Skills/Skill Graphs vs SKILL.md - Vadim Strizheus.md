# Skill Graphs vs SKILL.md

**Source:** [@VadimStrizheus](https://x.com/VadimStrizheus/status/2040687605600141619)
**Category:** Claude & Skills

## Résumé
Vadim découvre après 3 mois d'utilisation de Claude Code que les skill graphs offrent une approche supérieure aux simples fichiers SKILL.md. Heinrich argumente que les skill graphs permettent une meilleure organisation des connaissances structurées pour débloquer de nouvelles applications.

## Pourquoi c'est intéressant
Les skill graphs représentent une évolution significative dans la gestion des contextes et de la connaissance pour Claude. Cette découverte souligne l'importance d'explorer les fonctionnalités avancées au-delà des approches basiques.

## Points clés
- **Skill Graphs > SKILL.md:** Les skill graphs offrent une structure plus puissante que les fichiers SKILL.md statiques
- **Connaissance structurée:** L'organisation structurée de la connaissance déverrouille de nouvelles possibilités d'application
- **Apprentissage itératif:** Même après 3 mois d'utilisation, il y a encore des découvertes majeures à faire
