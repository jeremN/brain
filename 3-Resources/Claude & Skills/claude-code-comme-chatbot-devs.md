# 95% des devs utilisent Claude Code comme chatbot

**Source:** [@MikeCodeur on X](https://x.com/MikeCodeur/status/2018217862742372834)
**Category:** Claude & Skills

## Résumé
MikeCodeur observe que 95% des développeurs utilisent Claude Code comme un chatbot simple, plutôt que d'exploiter son potentiel complet. Il partage l'approche du créateur pour maximiser son utilisation.

## Pourquoi c'est intéressant
Cette statistique révèle un écart significatif entre l'utilisation réelle et le potentiel d'utilisation de Claude Code, avec des opportunités importantes pour améliorer la productivité des développeurs.

## Points clés
- **Usage gap** : Écart entre utilisation simple et potentiel réel
- **Creator's approach** : Meilleure façon d'utiliser Claude Code
- **Productivity potential** : Optimisation des workflows
