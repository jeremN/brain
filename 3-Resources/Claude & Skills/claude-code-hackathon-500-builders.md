# Claude Code Hackathon Wrap: 500 Builders, 1 Week

**Source:** [@claudeai on X](https://x.com/claudeai/status/2024986293248127452)
**Category:** Claude & Skills

## Résumé
Le hackathon Claude Code a rassemblé 500 builders pendant une semaine entière pour explorer et créer avec Claude Code. C'est un événement communautaire important démontrant l'enthousiasme et la capacité créative autour de Claude Code.

## Pourquoi c'est intéressant
Voir 500 développeurs travailler ensemble sur Claude Code crée une dynamique de découverte collective. Les projets et innovations issues d'une telle concentration de créativité fournissent des inspirations et patterns précieux.

## Points clés
- **500 builders** - Participation massive
- **1 week sprint** - Format intensif et productif
- **Community event** - Engagement et collaboration
- **Project showcase** - Démonstration de capacités réelles
