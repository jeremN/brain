# Ryan Singer's Shaping Skills for Claude Code

**Source:** [@rjs on X](https://x.com/rjs/status/2020184536194428951)
**Category:** Claude & Skills

## Résumé
Ryan Singer partage ses compétences et son approche du "shaping" appliquées à Claude Code. C'est une démonstration pratique de la façon de structurer et définir des projets dès leur conception.

## Pourquoi c'est intéressant
Les techniques de shaping de Ryan Singer, reconnues dans le product development, offrent un cadre systématique pour bien démarrer les projets avec Claude Code et éviter les approches chaotiques.

## Points clés
- **Shaping methodology** : Processus structuré de définition des projets
- **Priorisation** : Identifier ce qui compte vraiment au démarrage
- **Clarté** : Établir une vision claire avant de coder
