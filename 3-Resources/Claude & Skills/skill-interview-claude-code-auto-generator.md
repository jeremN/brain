# Skill /interview - Claude Code Auto-Interview Generator

**Source:** [@SamirBelabbes on X](https://x.com/SamirBelabbes/status/2013994129017929755)
**Category:** Claude & Skills

## Résumé
Un skill Claude Code innovant créé par Yann Decoopman qui génère automatiquement des entretiens d'embauche. Cet outil automatise la création et l'administration de processus d'entretien. Un exemple brillant de comment les skills peuvent résoudre des problèmes pratiques et répétitifs dans le processus de recrutement.

## Pourquoi c'est intéressant
L'automatisation intelligente des processus RH est très utile. Ce skill montre comment Claude Code peut être utilisé pour créer des outils pratiques qui économisent du temps tout en maintenant la qualité.

## Points clés
- **Automatisation des entretiens** - génération intelligente de questions
- **Skill réutilisable** - peut être adapté à différents contextes
- **RH optimization** - accélère le processus de recrutement
