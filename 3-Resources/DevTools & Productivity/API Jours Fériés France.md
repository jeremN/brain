# API Jours Fériés France

**Source:** [https://api.gouv.fr/documentation/jours-feries](https://api.gouv.fr/documentation/jours-feries)
**Category:** DevTools & Productivity
**Original Section:** APIs

## Summary
A REST API providing information about French public holidays (jours fériés). Useful for calendar applications, HR systems, and event scheduling in France.
## Why It's Interesting
Eliminates need to maintain hardcoded holiday data for France-based applications.
## Key Takeaways
- Free access to French holiday calendar data
- Covers national and regional holidays
- Easy integration with applications
- Useful for HR and scheduling tools
