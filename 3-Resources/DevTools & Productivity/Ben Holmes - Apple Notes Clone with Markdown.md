# Apple Notes Clone with Markdown

**Source:** [https://x.com/BHolmesDev/status/1972997511569920398](https://x.com/BHolmesDev/status/1972997511569920398)
**Category:** DevTools & Productivity

## Résumé
Ben Holmes explique comment construire un clone d'Apple Notes où la base de données est simplement du markdown. Cette approche permet une application de prise de notes fonctionnelle qui fonctionne sur plusieurs plateformes tout en gardant les données au format markdown simple.

## Pourquoi c'est intéressant
Utiliser markdown comme base de données simplifie dramatiquement l'architecture tout en offrant la portabilité et la flexibilité. C'est une approche brillante qui montre comment des solutions simples peuvent résoudre des problèmes complexes.

## Points clés
- **Markdown comme base de données** pour la simplicité
- **Architecture cross-platform** facile à maintenir
- **Données portables** et versionables avec git
- **Approche minimaliste** pour les applications de prise de notes
