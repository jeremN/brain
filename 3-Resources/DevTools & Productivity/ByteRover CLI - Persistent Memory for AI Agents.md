# ByteRover CLI — Persistent Structured Memory for AI Agents

**Source:** [https://github.com/campfirein/byterover-cli](https://github.com/campfirein/byterover-cli)
**Category:** DevTools & Productivity

## Summary
CLI fournissant une mémoire persistante et structurée pour les agents IA de coding. Organise le knowledge projet en arbre de contexte, synchronise en cloud, et distribue aux différents outils et membres d'équipe.

## Why It's Interesting
Résout le problème de perte de contexte entre sessions — 96.1% de précision sur les tests LoCoMo, avec support de 20+ LLM providers et 22+ agents IA.

## Key Takeaways
- **Interface REPL** : terminal React + Ink
- **24 outils built-in** : exécution de code, fichiers, mémoire
- **Cloud sync** : push/pull pour stockage distant
- **MCP compatible** : intégration Model Context Protocol
- **20+ LLM providers** : Anthropic, OpenAI, Google, etc.
- **4.3k GitHub stars**, Elastic License 2.0
- **Distribution** : binaires bundlés (pas besoin de Node.js) + npm
