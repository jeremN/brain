# Chrome Launcher

**Source:** [https://github.com/GoogleChrome/chrome-launcher](https://github.com/GoogleChrome/chrome-launcher)
**Category:** DevTools & Productivity
**Original Section:** Node

## Summary
An npm package for launching Chrome/Chromium browsers programmatically from Node.js. Used for headless browsing, testing, and automation tasks.
## Why It's Interesting
Foundation for browser automation and testing frameworks.
## Key Takeaways
- Programmatic browser control from Node.js
- Supports both headless and GUI modes
- Used by Puppeteer and other automation tools
- Handles browser lifecycle management
