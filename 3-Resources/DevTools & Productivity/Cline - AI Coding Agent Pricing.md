# Cline — AI Coding Agent Pricing

**Source:** [https://cline.bot/pricing](https://cline.bot/pricing)
**Category:** DevTools & Productivity

## Summary
Cline est un agent de coding IA autonome dans VS Code et CLI, open source. Modèle de pricing séparant les frais plateforme des coûts d'inférence (on paie le provider directement).

## Why It's Interesting
Alternative open-source à Cursor avec un modèle économique transparent — gratuit pour les individus, $20/user/mois pour les teams (10 premiers sièges gratuits à vie).

## Key Takeaways
- **Individuel** : gratuit avec ses propres clés API
- **Teams** : $20/user/mois (10 premiers sièges gratuits à vie)
- **Coût opérationnel** : $5-25/jour en usage intensif
- **Options modèles** : Gemini-2.0-pro (gratuit), DeepSeek-r1 ($0.65/M input), Claude 3.5/3.7 Sonnet ($3.00/M input)
- **VS Code + CLI** + JetBrains (Teams uniquement)
