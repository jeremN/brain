# Gemini CLI — Build, Debug & Deploy with AI

**Source:** [https://geminicli.com/](https://geminicli.com/)
**Category:** DevTools & Productivity

## Summary
Agent IA open-source de Google qui amène la puissance de Gemini directement dans le terminal. Utilise une boucle ReAct avec des outils built-in et des serveurs MCP pour accomplir des tâches complexes (bugs, features, tests).

## Why It's Interesting
L'équivalent Google de Claude Code — open-source (Apache 2.0), avec GEMINI.md, checkpointing, custom commands et intégration VS Code.

## Key Takeaways
- **Installation** : `npm install -g @google/gemini-cli`
- **Boucle ReAct** : reason + act avec outils locaux + MCP servers
- **GEMINI.md** : fichier de contexte projet (équivalent de CLAUDE.md)
- **Checkpointing** : sauvegarder et reprendre des conversations
- **Token caching** pour optimisation des coûts
- **Mode headless** pour scripting et automatisation
- **Open source** Apache 2.0
