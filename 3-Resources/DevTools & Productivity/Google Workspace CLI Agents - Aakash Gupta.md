# Google Workspace CLI avec Agents

**Source:** [@aakashgupta](https://x.com/aakashgupta/status/2029409062367199414)
**Category:** DevTools & Productivity

## Résumé
Google Workspace CLI par Addy Osmani offre l'accès à Google Drive, Gmail et Calendar via une interface en ligne de commande. Avec 40+ agent skills préconçus, c'est conçu pour fonctionner avec les humains et les agents d'IA. Avec 567 likes et 744 bookmarks, c'est un outil d'intégration puissant.

## Pourquoi c'est intéressant
L'approche de créer une CLI complète avec 40+ agent skills préconçus montre une compréhension profonde de comment les agents auront besoin d'interagir avec les outils professionnels. C'est une ressource critique pour automatiser les workflows Google Workspace.

## Points clés
- **Trois services intégrés**: Google Drive, Gmail, Calendar
- **40+ agent skills**: Couverture complète de fonctionnalités
- **Design dual**: Fonctionne avec humains et agents
