# Kitze — Developer, Entrepreneur & Educator

**Source:** [https://www.kitze.io/](https://www.kitze.io/)
**Category:** DevTools & Productivity

## Summary
Portfolio de Kitze — développeur, entrepreneur et éducateur. Créateur de Sizzy (browser responsive), Kitze UI (design system shadcn), template Next.js SaaS, et cours interactifs React/GraphQL.

## Why It's Interesting
Écosystème complet d'un indie dev prolifique — des outils dev aux cours en passant par un "life operating system" personnel.

## Key Takeaways
- **Sizzy** : browser dev qui affiche tous les layouts côte à côte
- **Kitze UI** : design system basé sur shadcn avec thèmes et patterns prêts à l'emploi
- **Template Next.js** : auth, paiements, UI pré-configurés pour lancer un SaaS rapidement
- **Cours** : workshops React, GraphQL, full-stack
- **Life OS** : système d'organisation personnelle tout-en-un
