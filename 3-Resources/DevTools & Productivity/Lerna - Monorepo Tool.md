# Lerna - Monorepo Tool

**Source:** [https://lerna.js.org/](https://lerna.js.org/)
**Category:** DevTools & Productivity
**Original Section:** To Classify

## Summary
A popular tool for managing JavaScript monorepos (multiple packages in single repository). Handles versioning, dependency management, and publishing across packages.
## Why It's Interesting
Enables scalable JavaScript project organization at enterprise scale.
## Key Takeaways
- Simplifies monorepo management
- Handles versioning and publishing
- Manages cross-package dependencies
- Widely adopted in large projects
