# Prisma ORM

**Source:** [https://www.prisma.io/](https://www.prisma.io/)
**Category:** DevTools & Productivity
**Original Section:** To Classify

## Summary

Next-generation Node.js/TypeScript ORM with type-safe database access, auto-generated migrations, and visual database browser. Schema defined in declarative Prisma Schema Language generates TypeScript types, SQL migrations, and typed query client. Supports PostgreSQL, MySQL, SQLite, MongoDB.

## Why It's Interesting

Replaced manual SQL and traditional ORMs in most new TypeScript/Node.js projects

## Key Takeaways

- Schema-first: define data model once, get types, migrations, and client auto-generated
- Fully type-safe queries: impossible to write queries that don't match your schema
- Prisma Studio: visual database browser for exploring/editing data
- Prisma Migrate: declarative migration system generating SQL from schema changes
- Supports relations, raw SQL, transactions, and middleware
