# QMD - Query Markup Documents

**Source:** [https://github.com/tobi/qmd](https://github.com/tobi/qmd)
**Category:** DevTools & Productivity

## Summary
A markup language for documents that supports queries and dynamic content. Enables documents to reference and query external data sources.
## Why It's Interesting
Bridges static documents and dynamic data sources.
## Key Takeaways
- Markup language with query capabilities
- Dynamic document content generation
- Useful for generated documentation
- Separates content from presentation
