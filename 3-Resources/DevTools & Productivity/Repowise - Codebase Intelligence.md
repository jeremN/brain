# Repowise — Codebase Intelligence for AI-Assisted Teams

**Source:** [https://github.com/repowise-dev/repowise](https://github.com/repowise-dev/repowise)
**Category:** DevTools & Productivity

## Summary
Plateforme d'intelligence de codebase pour équipes d'ingénierie assistées par IA. Indexe quatre couches (graphe de dépendances, historique git, documentation auto-générée, décisions architecturales) et les expose via un serveur MCP compatible Claude Code.

## Why It's Interesting
Comble le gap entre le code brut et le contexte dont les agents IA ont besoin — documentation auto-maintenue, détection de code mort, et analytics git, le tout accessible via MCP.

## Key Takeaways
- **8 outils MCP** : contexte, analyse de risques, recherche sémantique, code mort, décisions architecturales
- **Intelligence Git** : hotspots, propriété du code, co-changements, facteur bus
- **Doc auto-générée** : wiki avec scores de fraîcheur
- **CLAUDE.md auto-généré** : résumé architectural + décisions actives
- **Stack** : Python (tree-sitter, NetworkX, LanceDB), React, Docker
