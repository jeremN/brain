# VPS Hetzner Config Setup - Simony

**Source:** [@0xSimony](https://x.com/0xSimony/status/2039321643084865874)
**Category:** DevTools & Productivity

## Résumé
Configuration d'un VPS Hetzner CX23 avec 2 vCPU, 4 Go de RAM et 40 Go de SSD pour un coût mensuel de 4,19€. Cette solution est idéale pour tester et déployer des applications légères.

## Pourquoi c'est intéressant
Hetzner offre un excellent rapport qualité-prix pour les développeurs souhaitant avoir une infrastructure de test à faible coût. C'est une alternative viable aux options cloud plus onéreuses pour les petits projets.

## Points clés
- **Spécifications:** 2 vCPU, 4 Go RAM, 40 Go SSD
- **Prix:** 4,19€/mois
- **Cas d'usage:** Testing et déploiement d'applications
