# CSS Container Queries

**Source:** [https://www.oddbird.net/2022/08/18/cq-syntax/](https://www.oddbird.net/2022/08/18/cq-syntax/)
**Category:** Frontend & Web Dev
**Original Section:** CSS

## Summary

OddBird's guide on CSS Container Queries — the feature that lets components respond to their container's size rather than the viewport. Explains @container rules, container-type property, container-name, and new cqw/cqh units. Enables truly reusable responsive components.

## Why It's Interesting

The most significant CSS layout feature since Flexbox/Grid — enables component-level responsive design

## Key Takeaways

- Container queries let components respond to parent container size instead of viewport
- Set container-type: inline-size on the parent, use @container on children
- Named containers with container-name allow targeting specific ancestors
- New units: cqw (container query width), cqh (height), cqi (inline), cqb (block)
- Enables truly portable, context-aware components regardless of placement
