# Sensible img Defaults

**Source:** [https://x.com/csswizardry/status/1961105967686594577](https://x.com/csswizardry/status/1961105967686594577)
**Category:** Frontend & Web Dev

## Résumé
CSS Wizardry partage img.css, un fichier de styles CSS proposant des valeurs par défaut sensées pour les éléments img. Ce projet fournit une base solide pour styliser les images de manière cohérente et accessible dans les projets web.

## Pourquoi c'est intéressant
Plutôt que de réinventer la roue à chaque projet, disposer d'une collection de defaults CSS testés et bien pensés pour les images accélère le développement et assure une meilleure cohérence dans les projets.

## Points clés
- **Defaults CSS pour éléments img** testés et éprouvés
- **Approche accessible** pour le traitement des images
- **Base réutilisable** entre les projets
- **Disponible sur GitHub** pour la communauté
