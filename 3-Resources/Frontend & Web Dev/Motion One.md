# Motion One

**Source:** [https://motion.dev/guides/quick-start](https://motion.dev/guides/quick-start)
**Category:** Frontend & Web Dev
**Original Section:** To Classify

## Summary

Motion (formerly Framer Motion) is a production-grade animation library for React, JavaScript, and Vue. Created by Matt Perry, it's the fastest-growing animation library with 18M+ monthly npm downloads. Features include: simple declarative API, independent transform animations, hardware-accelerated scroll animations, exit animations with AnimatePresence, native hover/press/drag gestures, layout animations, timeline sequences, and spring physics. Motion+ offers premium components (magnetic cursors, infinite tickers), 330+ pre-built animations, and an AI kit with context rules.

## Why It's Interesting

The dominant web animation library with a growing AI integration story — the AI kit for LLM context is a clever product move

## Key Takeaways

- Production-grade animation for React, JS, and Vue — 18M+ monthly npm downloads, surpassing GSAP and React Spring
- Simple API: animate transforms independently without wrapper elements, spring physics for natural-feeling motion
- Hardware-accelerated scroll animations and exit animations with AnimatePresence component
- Layout animation engine: animate between different layouts with industry-leading performance
- Motion+ premium: 330+ pre-built animations, AI kit with LLM context rules, transition editor for IDEs
- Great for LLMs: includes an AI kit with docs and source code context for accurate AI-generated animations
