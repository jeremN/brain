# Radix UI

**Source:** [https://www.radix-ui.com/](https://www.radix-ui.com/)
**Category:** Frontend & Web Dev
**Original Section:** Libraries / UI

## Summary

Unstyled, accessible UI component primitives for React design systems. Provides fully accessible dialogs, dropdowns, tabs, tooltips etc. with no visual styling — you add appearance. WAI-ARIA compliant, keyboard navigable, screen reader friendly. Powers shadcn/ui.

## Why It's Interesting

The headless UI approach revolutionized how design systems are built

## Key Takeaways

- Unstyled by design: provides behavior and accessibility, you add the styling
- Full WAI-ARIA compliance and keyboard navigation built in
- Foundation for shadcn/ui and many enterprise design systems
- Handles complex interactions: focus trapping, dismiss on escape, portal rendering
- Composable API: use only the parts you need
