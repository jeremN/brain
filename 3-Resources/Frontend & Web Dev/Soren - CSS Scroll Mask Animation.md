# CSS Scroll Mask Animation

**Source:** [https://x.com/sorenblank/status/1982502993237832185](https://x.com/sorenblank/status/1982502993237832185)
**Category:** Frontend & Web Dev

## Résumé
Soren démontre comment créer une animation de masquage CSS synchronisée avec le scroll en utilisant la propriété animation-timeline. Cette technique utilise .masked-overflow pour contrôler le masquage du contenu en fonction du défilement de l'utilisateur.

## Pourquoi c'est intéressant
L'animation-timeline CSS est une fonctionnalité puissante et relativement nouvelle qui permet de lier les animations directement au scroll. Cette approche offre des performances supérieures par rapport aux solutions JavaScript traditionnelles.

## Points clés
- **animation-timeline CSS** pour synchroniser les animations au scroll
- **Classe .masked-overflow** pour gérer le masquage du contenu
- **Performance optimale** sans recalculs JavaScript complexes
