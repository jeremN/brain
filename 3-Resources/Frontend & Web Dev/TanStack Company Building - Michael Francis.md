# Nous N'Avons Pas Basculé vers TanStack. Nous Avons Construit Une Compagnie Sur Elle.

**Source:** [@MFrancis107](https://x.com/MFrancis107/status/2028869327726723463)
**Category:** Frontend & Web Dev

## Résumé
TanStack n'est pas une simple plateforme de vibe coding, c'est une plateforme d'ingénierie complète sur laquelle une compagnie peut être construite. Michael Francis explique comment TanStack fournit les fondations nécessaires pour bâtir une infrastructure d'ingénierie professionnelle et scalable.

## Pourquoi c'est intéressant
La distinction entre une "plateforme de vibe coding" et une "plateforme d'ingénierie" est importante. Elle montre que TanStack offre de la profondeur architecturale au-delà de la génération de code rapide, ce qui est essentiellement pour les organisations sérieuses.

## Points clés
- **Plateforme d'ingénierie**: Bien plus qu'un outil de vibe coding
- **Culture d'entreprise**: Fondations pour construire une compagnie
- **Scalabilité**: Infrastructure pour la croissance long terme
