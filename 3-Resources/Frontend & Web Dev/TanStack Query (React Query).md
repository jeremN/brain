# TanStack Query (React Query)

**Source:** [https://tanstack.com/query/v4/](https://tanstack.com/query/v4/)
**Category:** Frontend & Web Dev
**Original Section:** Libraries

## Summary

Industry-standard library for managing async server state in React and other frameworks. Handles data fetching, caching, synchronization, background updates, pagination, and optimistic updates with minimal boilerplate.

## Why It's Interesting

Essential library that replaced hundreds of lines of manual data-fetching boilerplate

## Key Takeaways

- Server state and client state are fundamentally different — TanStack Query manages server state properly
- Automatic background refetching keeps data fresh without manual polling
- Built-in cache management with configurable stale times reduces network requests
- Handles pagination, infinite scroll, and optimistic updates out of the box
- Framework-agnostic: works with React, Vue, Solid, Svelte, Angular
