# The 49MB Web Page — News Site Audit

**Source:** [https://thatshubham.com/blog/news-audit](https://thatshubham.com/blog/news-audit)
**Category:** Frontend & Web Dev

## Summary
Audit architectural d'un site d'actualités montrant l'ampleur du tracking, des ads programmatiques et du poids des pages. Plaidoyer pour un web plus rapide, respectueux des utilisateurs, sans paywalls ni surveillance.

## Why It's Interesting
Démontage méthodique de l'économie toxique des sites d'actualités — les bidding requests aux ad exchanges partent avant même que l'utilisateur ait fini de lire le titre. Illustre le cercle vicieux CPM/rétention.

## Key Takeaways
- **Tracking massif** : requêtes vers Rubicon Project, Amazon Ad Systems dès le chargement
- **Cookie banners = boucliers juridiques** pour protéger les éditeurs pendant le data mining
- **Spirale mortelle du programmatic ad-tech** : revenus CPM court-terme vs rétention long-terme
- **Plaidoyer** : pas de paywalls, pas d'ads programmatiques, pas d'appareil de surveillance
