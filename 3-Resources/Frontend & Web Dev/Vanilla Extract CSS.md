# Vanilla Extract CSS

**Source:** [https://formidable.com/blog/2021/vanilla-extract/](https://formidable.com/blog/2021/vanilla-extract/)
**Category:** Frontend & Web Dev
**Original Section:** Libraries / CSS

## Summary

CSS-in-TypeScript library generating static CSS at build time with zero runtime. Write styles in .css.ts files with full TypeScript type safety. Supports theming, variants, responsive styles, and all major bundlers.

## Why It's Interesting

Best of both worlds: CSS-in-JS developer experience with static CSS performance

## Key Takeaways

- Zero runtime: all CSS extracted at build time, unlike styled-components or Emotion
- Full TypeScript integration: styles are type-checked and refactorable
- Sprinkles API: type-safe utility-class system similar to Tailwind but with TypeScript
- Theme contracts: type-safe theming across entire application
- Co-locate styles with components while still getting static CSS output
