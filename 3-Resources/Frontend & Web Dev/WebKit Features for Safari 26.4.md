# WebKit Features for Safari 26.4

**Source:** [https://webkit.org/blog/17862/webkit-features-for-safari-26-4/](https://webkit.org/blog/17862/webkit-features-for-safari-26-4/)
**Category:** Frontend & Web Dev

## Summary
Safari 26.4 (mars 2026) apporte 44 nouvelles fonctionnalités et 191 correctifs. Les trois headliners : CSS Grid Lanes (layouts masonry natifs), WebTransport (alternative moderne à WebSocket), et Keyboard Lock API.

## Why It's Interesting
CSS Grid Lanes est LA feature attendue depuis des années pour le masonry layout natif. WebTransport ouvre la porte aux expériences low-latency (jeux multijoueurs, collaboration temps réel).

## Key Takeaways
- **CSS Grid Lanes** : layouts masonry natifs en CSS Grid — galeries visuelles sans JS
- **WebTransport** : alternative à WebSocket pour jeux, collab live, visioconférence
- **Keyboard Lock API** : meilleur contrôle des raccourcis clavier pour les web apps
- **Focus stabilité** : résultat des surveys développeurs 2025 demandant du temps pour rattraper les features existantes
- **191 bug fixes** en plus des 44 features
