# Zod

**Source:** [https://zod.dev/](https://zod.dev/)
**Category:** Frontend & Web Dev
**Original Section:** Libraries / JS

## Summary

TypeScript-first schema declaration and validation library with automatic static type inference. Declare a schema once and get both runtime validation and TypeScript types — eliminating drift between types and validation. Deep ecosystem integration with tRPC, React Hook Form, Prisma, Next.js.

## Why It's Interesting

Became the standard for runtime type validation in TypeScript, replacing Joi/Yup

## Key Takeaways

- Single source of truth: define schema once, get both runtime validation and TypeScript types
- Composable: combine schemas with .merge(), .extend(), .pick(), .omit(), .partial()
- Rich error messages with .refine() and .superRefine() for custom validation
- Works with tRPC, React Hook Form, Prisma, Next.js — deep TS ecosystem integration
- Zero dependencies, tiny bundle, works in Node.js and browsers
