# Animation d'Accordéon en CSS avec interpolate-size

**Source:** [@vponamariov](https://x.com/vponamariov/status/2031310016251310530)
**Category:** Frontend & Web Dev

## Résumé
Victor annonce qu'une seule propriété CSS, `interpolate-size`, a éliminé le besoin de JavaScript dans chaque FAQ accordéon. Cette propriété CSS permet des animations fluides et automatiques pour les accordéons sans dépendre d'une logique JavaScript complexe.

## Pourquoi c'est intéressant
C'est un exemple parfait de comment les avancées CSS natives éliminent progressivement la dépendance à JavaScript pour des interactions courantes. Cela simplifie le code, réduit la charge JavaScript et améliore les performances.

## Points clés
- **CSS pur** : Pas de JavaScript nécessaire pour les accordéons
- **Animations fluides** : Transitions automatiques et lisses
- **Propriété interpolate-size** : Outil CSS natif pour les animations de taille
