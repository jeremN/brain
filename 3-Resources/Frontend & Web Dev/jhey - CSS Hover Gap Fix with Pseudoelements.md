# CSS Hover Gap Fix with Pseudoelements

**Source:** [https://x.com/jh3yy/status/1980360248562184498](https://x.com/jh3yy/status/1980360248562184498)
**Category:** Frontend & Web Dev

## Résumé
Jhey présente une technique pour combler l'écart qui apparaît lors du survol d'éléments en utilisant des pseudoéléments CSS combinés avec :has() et le positionnement d'ancrage. Cette solution résout un problème courant d'UX où les utilisateurs perdent l'état de survol lors du passage entre éléments.

## Pourquoi c'est intéressant
Ce détail d'implémentation montre comment des fonctionnalités CSS modernes comme :has() et anchor-positioning peuvent résoudre des problèmes d'interaction utilisateur. C'est une preuve de l'importance des petits détails qui améliorent l'expérience globale.

## Points clés
- **Pseudoéléments pour combler les écarts** entre éléments survolés
- **Sélecteur :has()** pour détecter les états de survol
- **Anchor positioning** pour les indicateurs de pagination
- **Amélioration d'UX** sans JavaScript complexe
