# Fix CSS text-box-trim pour l'Espacement du Texte

**Source:** [@bossadizenith](https://x.com/bossadizenith/status/2031373257157320939)
**Category:** Frontend & Web Dev

## Résumé
Zenith partage une solution CSS en une ligne pour corriger l'espacement étrange qui apparaît naturellement au-dessus des majuscules et en dessous de la ligne de base du texte. La propriété `text-box-trim` élimine cet espacement indésirable et améliore la précision typographique des designs web.

## Pourquoi c'est intéressant
C'est un problème de typographie que chaque développeur frontend a rencontré mais que peu savaient comment résoudre élégamment. Cette solution CSS native est beaucoup plus simple que les hacks habituels et améliore la qualité visuelle des designs.

## Points clés
- **Une ligne CSS** : Solution minimale et élégante
- **Espacement vertical** : Corriger l'espacement au-dessus et en dessous du texte
- **Propriété text-box-trim** : Outil natif pour la précision typographique
