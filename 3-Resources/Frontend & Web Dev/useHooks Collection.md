# useHooks Collection

**Source:** [https://usehooks.com/](https://usehooks.com/)
**Category:** Frontend & Web Dev
**Original Section:** React / Hooks

## Summary

Curated collection of reusable, well-documented React hooks for common use cases. Each hook comes with TypeScript types, usage examples, and SSR compatibility. Covers local storage, media queries, clipboard, debounce, intersection observer, dark mode, and more.

## Why It's Interesting

Go-to reference for React hooks patterns, saving hours of reinventing common abstractions

## Key Takeaways

- Ready-to-use hooks eliminate writing common utility hooks from scratch
- Full TypeScript support with proper generic types
- SSR-safe implementations for Next.js and server-rendering frameworks
- Each hook independently importable — no need to install the whole library
- Well-documented with interactive examples and edge case handling
