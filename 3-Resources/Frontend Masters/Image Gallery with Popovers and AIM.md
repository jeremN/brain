# Image Gallery with Popovers and AIM (Anchor-Interpolated Morph)

**Source:** [https://frontendmasters.com/blog/image-gallery-with-popovers-and-aim-anchor-interpolated-morph/](https://frontendmasters.com/blog/image-gallery-with-popovers-and-aim-anchor-interpolated-morph/)
**Category:** Frontend Masters

## Summary
Tutoriel CSS pur pour créer une galerie d'images avec ouverture en grand via popovers, animations d'ancrage (AIM), et zéro JavaScript. Les miniatures servent d'ancres CSS, et les versions agrandies s'animent depuis/vers ces ancres.

## Why It's Interesting
Démo impressionnante des capacités CSS modernes — galerie photo avec agrandissement animé, entièrement sans JS, grâce à popover, anchor positioning et @starting-style.

## Key Takeaways
- **Popover sur image** : attribut `popover` directement sur l'élément image après un bouton
- **`transition-behavior: allow-discrete`** : permet les animations malgré le changement de `display`
- **`@starting-style`** + `:popover-open`** : animations d'entrée/sortie fluides
- **Anchor positioning** : les miniatures deviennent des ancres CSS
- **Zero JS** : tout est fait en CSS pur
