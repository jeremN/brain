## Summary
Outil web qui transforme le graphe de contributions GitHub en un GIF animé de jeu Space Shooter. Disponible en interface web, CLI, et GitHub Action.

Projet fun et créatif — parfait pour un README GitHub original. Plusieurs stratégies d'animation disponibles (colonne, ligne, random).
