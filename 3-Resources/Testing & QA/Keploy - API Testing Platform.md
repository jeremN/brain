# Keploy - API Testing Platform

**Source:** [github.com/keploy/keploy](https://github.com/keploy/keploy)
**Category:** Testing & QA

## Summary
Keploy is an open-source platform for API testing and mocking. It records real API traffic and generates test cases automatically, enabling efficient testing without manual test case creation.

## Why It's Interesting
Automates API testing through traffic recording, reducing manual testing overhead. Particularly valuable for teams adopting AI-driven development workflows.

## Key Takeaways
- Automatic test case generation from traffic
- API mocking and recording capabilities
- Reduces testing overhead
