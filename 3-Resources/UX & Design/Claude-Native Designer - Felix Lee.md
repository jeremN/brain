# Le Designer Claude-Native

**Source:** [@witcheer](https://x.com/witcheer/status/2029279217020436753)
**Category:** UX & Design

## Résumé
Felix Lee, designer avec 7 années d'expérience, propose une nouvelle approche: cesser d'utiliser Figma exclusivement et livrer des produits directement via Claude Code. L'article "The Claude-Native Designer" remet en question le workflow de design traditionnel en montrant comment Claude Code peut être utilisé pour la création de produits plus rapide.

## Pourquoi c'est intéressant
Cette perspective représente une remise en question fondamentale des workflows de design modernes. Pour les designers qui adoptent Claude Code, cela offre un chemin alternatif vers la livraison de produits qui pourrait être plus efficace que les outils de design traditionnels.

## Points clés
- **Approche alternative**: Au-delà des outils de design traditionnel
- **Claude Code pour le design**: Utiliser Claude Code pour livrer des produits
- **Expérience reconnue**: Conseils d'un designer avec 7 ans d'expérience
