# Mobbin

**Source:** [https://mobbin.com/](https://mobbin.com/)
**Category:** UX & Design

## Summary

Mobbin is the leading UI/UX design reference platform with 1,150+ real-world iOS and web apps, 602,300+ screens, and 321,100+ user flows. Designers search by screen type (Profile, Wallet, Checkout, etc.) to find real design patterns in seconds. Key features: full user flow videos with transitions and micro-interactions, interactive prototype mode, copy-to-Figma plugin, collections for organizing references, and commenting. Used by design teams at Figma, Shopify, Visa, and Plaid.

## Why It's Interesting

The definitive UI pattern reference — replaces the chaos of unorganized screenshots with searchable, categorized real-world examples

## Key Takeaways

- Library of 1,150+ real apps with 602K+ screens and 321K+ flows — the largest curated UI reference database
- Search by screen type (Profile, Checkout, Settings, etc.) or text within screenshots to find patterns instantly
- Full user flow videos show transitions, micro-interactions, and animations in context — not just static screenshots
- Figma plugin lets you copy designs directly into your workspace for reference
- Free tier available with paid plans for full access — used by teams at Figma, Shopify, Visa, and Plaid
