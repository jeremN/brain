# Nord Health Design System

**Source:** [https://nordhealth.design/](https://nordhealth.design/)
**Category:** UX & Design
**Original Section:** Design System

## Summary

Nord is Nordhealth's open design system for building coherent healthcare software experiences. It provides design tokens (platform-agnostic style variables), Web Components (framework-agnostic UI components with encapsulated styles), a utility-centric CSS framework, powerful theming with pre-made themes, Nordicons icon set, and a Figma toolkit for prototyping. Fully open source on GitHub.

## Why It's Interesting

Well-executed healthcare design system using Web Components — good reference for building framework-agnostic component libraries

## Key Takeaways

- Framework-agnostic: uses Web Components with encapsulated styles that work with any frontend framework
- Design tokens as platform-agnostic variables ensure consistent look and feel across platforms
- Utility-centric CSS framework provides building blocks without opinionated component styles
- Includes Figma toolkit for designer-developer handoff and prototyping
- Open source on GitHub with Slack channel for community feedback
