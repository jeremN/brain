# Spline - 3D Design Tool

**Source:** [https://spline.design/](https://spline.design/)
**Category:** UX & Design
**Original Section:** To Classify

## Summary

Spline is a browser-based collaborative 3D design platform for creating interactive, production-ready experiences in real-time. Features include: familiar design tools for 3D modeling, real-time rendering, layer-based materials, interactive event systems (states, events, game controls, physics, particles), timeline animation, and variables/data integration via webhooks and APIs. Ships to Web, iOS, and Android with framework integrations for React, Next.js, Webflow, Framer, and more.

## Why It's Interesting

Democratizes interactive 3D for the web — designers can build production-ready 3D experiences without developer handoff

## Key Takeaways

- Browser-based 3D design with real-time collaboration — no desktop app required
- Built-in interactivity: states, events, game controls, physics, and particle systems
- Multi-platform export: web (HTML/JS, React, Next.js, Webflow, Framer, Wix), iOS (Swift), Android (Kotlin)
- Variables and data integration via Webhooks and APIs — bring real-time data into 3D experiences
- Includes Hana: a 2D collaborative design tool with auto layout, vector networks, and visual effects
- Free to start, used by teams at major organizations for interactive brand experiences
