# How to Vibe Code Beautiful UI: Sketch First, Prompt Second

**Source:** [@om_patel5 on X](https://x.com/om_patel5/status/2025705776434520284)
**Category:** UX & Design

## Résumé
Om Patel explique comment créer une belle UI via vibe-coding avec une approche sketch-first utilisant Excalidraw avant les prompts. C'est une méthodologie pratique combinant la conception manuelle et l'IA.

## Pourquoi c'est intéressant
Le workflow sketch-first crée une communication plus claire entre l'intention du designer et l'IA, résultant en des interfaces plus fidèles à la vision originale. C'est une approche hybride puissante et efficace.

## Points clés
- **Sketch first** - Élaboration initiale manuelle
- **Excalidraw** - Outil simple pour croquis
- **Prompt second** - Traduction en code AI
- **Vibe coding** - Approche intuitive et créative
